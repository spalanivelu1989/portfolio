# CONSTITUTION.md
# Rules this project never breaks

1. **No untested code in production** — every feature, component, or page must have tests (unit or integration) before it ships.
2. **Content is king** — the site must load fast and be readable without JavaScript enabled where possible.
3. **Accessible by default** — all interactive elements must be keyboard-navigable and screen-reader friendly (WCAG 2.1 AA minimum).
4. **No feature creep** — if a feature doesn't serve the goal of impressing or informing a potential employer or client, it doesn't belong here.

---
*Add, edit, or remove rules here as the project evolves. Keep it short — these must be quick to read.*
